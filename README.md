# Updrafts
Different Gaussian Process Emulators (GPEs) for predicting updraft velocities.

Emulators are trained based on UCLALES-SALSA (https://github.com/UCLALES-SALSA) model predictions. Detailed description of each emulator is given in the corresponding directory.

See also https://github.com/kallenordling/eclair_emulator and https://github.com/JaakkoAhola/DESIGN for the previous updraft velocity emulator versions.
